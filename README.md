### App. Functionalities
* <strong>GET</strong> a list of users in a table
* <strong>CREATE</strong> a new user 
* <strong>EDIT</strong> any user field
* <strong>Sort</strong> users by id
* <strong>DELETE</strong> a desired user from the table
